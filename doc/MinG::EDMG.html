INTERNAL CLASSES AND FUNCTIONS

other_p:(Int $i --> Int): Subroutine that returns 0 if it gets 1, and 1 otherwise.

EXPORTED CLASSES AND FUNCTIONS

MinG::EDMG::Feature: Class that defines an EDMG feature.

from_str:(MinG::EDMG::Feature $: Str $s, *%_ --> MinG::EDMG::Feature): Method that returns an EDMG Feature out of a string description.

MinG::EDMG::LItem: Class that defines an EDMG lexical item.

MinG::EDMG::Grammar: Class that defines an EDMG grammar.

MinG::EDMG::Derivation: Class that implements a Derivation for our EDMG parser.

scan:(MinG::EDMG::Derivation $: QueueItem $pred, Int $child_place, *%_ --> MinG::EDMG::Derivation): See Stabler (2013)
merge2:(MinG::EDMG::Derivation $: QueueItem $pred, Node @non_terms, Node $selected, Node $selector, *%_ --> MinG::EDMG::Derivation): See Stabler (2013)
merge3:(MinG::EDMG::Derivation $: QueueItem $pred, Node @leaves, Node $mover_child, Node $selector, Mover $mover, *%_ --> MinG::EDMG::Derivation): See Stabler (2013)
merge4:(MinG::EDMG::Derivation $: QueueItem $pred, Node @non_terms, Node $mover_child, Node $selector, Mover $mover, *%_ --> MinG::EDMG::Derivation): See Stabler (2013)
move1:(MinG::EDMG::Derivation $: QueueItem $pred, Node $licensor, Node $licensed, *%_ --> MinG::EDMG::Derivation): See Stabler (2013)
move2:(MinG::EDMG::Derivation $: QueueItem $pred, Node $licensor, Node $mover, Node $mover_child, *%_ --> MinG::EDMG::Derivation): See Stabler (2013)

MinG::EDMG::Parser: Class that runs a Stabler (2013)-type parser for EDMGs.

devq_to_str:(MinG::EDMG::Parser $: *%_): Method that gets a nice string representation of the @!devq.
parallel_run:(MinG::EDMG::Parser $: *%_ --> Bool): Method that runs one iteration of the parsing loop, running one step of each derivation in parallel. Gets all possible derivations.
parallel_parse:(MinG::EDMG::Parser $: *%_ --> Bool): Method that runs the main parsing loop using parallel_run. Gets all possible derivations.
parse_str:(MinG::EDMG::Parser $: Str $inp, ParseWay $do = ParseWay::PARALLEL, Str $compile = "", *%_ --> Bool): Method that parses a single string based on the grammar that was initialised using Parser.init()
init:(MinG::EDMG::Parser $: MinG::EDMG::Grammar $g, *%_): Method that initialises that parser to later parse several strings. Should be used instead of Parser.setup when a single grammar is going to be used several times.
re_init:(MinG::EDMG::Parser $: *%_): Method that re-initialises the parser using the full version of the grammar.
large_parse:(MinG::EDMG::Parser $: Str $inp, ParseWay $do = ParseWay::PARALLEL, Str $compile = "", *%_ --> Bool): Method that deletes all non-phonetically-empty words that don't appear in the input before parsing. When using large grammars, this can be much more efficient, but has a large constant time-cost, so it will make small grammars slower.
